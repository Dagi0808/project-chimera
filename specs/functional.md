## Agent Responsibilities

### Trend Discovery
- As an Agent, I need to fetch current content trends from multiple platforms.
- As an Agent, I need to normalize trend data into a common schema.
- As an Agent, I need to rank trends based on engagement velocity.

---

### Content Safety
- As an Agent, I must validate generated content against safety policies.
- As an Agent, I must flag high-risk content for human review.
- As an Agent, I must block publishing if safety validation fails.

---

### Content Generation
- As an Agent, I need to generate content ideas based on trend inputs.
- As an Agent, I need to generate captions and metadata.
- As an Agent, I need to associate generated content with its originating trend.

---

### Publishing & Engagement
- As an Agent, I need to publish content to supported platforms.
- As an Agent, I need to record publishing outcomes.
- As an Agent, I need to track engagement metrics over time.

---

### Agent Network Participation
- As an Agent, I need to advertise my availability and capabilities.
- As an Agent, I need to accept tasks from other agents.
- As an Agent, I need to report task execution status.
